# Frontend

Cliente web estatico del proyecto.

## Contenido

- `.env.example`: variables publicas futuras del cliente.
- `index.html`: landing principal.
- `analisis.html`: formulario para subir imagen.
- `assets/css`: estilos.
- `assets/js`: logica del cliente.
- `assets/images`: imagenes.
- `assets/models`: modelos locales heredados.

## Nota

El frontend muestra un rostro de referencia y accesorios sugeridos desde el catalogo local. Un archivo `frontend/.env` no se consume automaticamente en navegador sin una etapa de build o inyeccion de configuracion publica.
