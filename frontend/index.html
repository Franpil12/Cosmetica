Beneficios

Una experiencia mas clara, util y facil de usar.

01

Personalizacion real

Las sugerencias no salen de una plantilla generica. Se apoyan en la lectura del rostro y en el catalogo del proyecto.

02

Resultados entendibles

La app muestra explicacion, accesorios sugeridos y senales usadas para llegar a la recomendacion final.

03

Privacidad practica

El flujo se centra en procesar la imagen para el analisis y devolver el resultado sin exponer una red social ni perfiles publicos.

04

Diseno adaptable

Todo el frontend se reorganiza correctamente en telefono, tablet y escritorio sin romper formularios ni resultados.

Proceso

Como funciona el analisis

1

Sube una selfie frontal

La mejor entrada es una foto con buena luz, rostro visible completo y fondo simple.

2

La API analiza la imagen

El backend combina la lectura de OpenAI con el clasificador local para estimar la forma del rostro.

3

Recibes referencias y accesorios

El resultado muestra rostro de referencia, foco de accesorios y recomendaciones visuales del catalogo.

Mision

Belleza en Codigo busca convertir un analisis tecnico en una experiencia simple para cualquier persona.

El proyecto cruza vision computacional, catalogos visuales y una interfaz clara para sugerir opciones esteticas que tengan contexto y no solo un resultado aislado.

Valores

  • Transparencia sobre como se genera cada sugerencia.
  • Accesibilidad visual en escritorio y movil.
  • Base tecnica preparada para desplegar frontend y backend por separado.
  • Mejora continua del catalogo y de la experiencia de usuario.
